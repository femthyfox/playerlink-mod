# Player-Linked Redstone Frequencies

A NeoForge 1.21.1 addon for the **Create** mod (6.0.x) that makes Redstone
Link frequencies **player-specific** so that two unrelated players on the
same multiplayer server using the same item combination won't
cross-connect.

## Features

**multiplayer.** you can now use redstonelinks to their full ability on multiplayer servers without having to sacrifice blocks 



**singleplayer.** you now have access to double the items due to the unlinked links and the player linked alternative
